# Install and run
```bash
$ mkdir -p ~/dev/ti
$ cd ~/dev/ti
$ git clone https://github.com/bom4v/ti-data-samples.git
$ git clone https://github.com/telecoms-intelligence/induction-anonymization.git
$ cd induction-anonymization
$ ./mkLocalDir.sh
$ ./python/anonymization.py
```
